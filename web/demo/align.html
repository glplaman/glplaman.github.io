水平居中
.文字/inline/inline-block
text-align: center;
单行文本-水平垂直居中
.行高和高度保持一致；如果用到背景变化，在高度比较大的情况下，推荐使用flex
.行高可以被继承，所以要小心使用
heigh: 60px;
line-height: 60px;
text-align: center;
单个元素-水平垂直居中
.容器如果有宽度和高度，效果更直观
1.弹性盒子-只操作容器
.wrap {
    display: flex;
    justify-content: center;
    align-items: center;
}
2.弹性盒子-同时操作容器和子元素
        .wrap {
          display: flex;//core
          width: 300px;
          height: 400px;
        }
    
        .wrap div {
          margin: auto;//core
          width: 80px;
          height: 60px;
        }
3.定位
.父相子绝
.变形transform
.子元素甚至可以大过容器尺寸
        .wrap {
          position: relative;
          width: 300px;
          height: 400px;
          background-color: #f40;
        }
    
        .wrap div {
          position: absolute;
          left: 50%;
          top: 50%;
          transform: translate(-50%, -50%);
          width: 80px;
          height: 60px;
        }
