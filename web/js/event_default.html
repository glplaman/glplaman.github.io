默认事件

@Event Default
常见的默认事件有
<a> 的默认跳转
<form> 的默认提交跳转
右键菜单
[] 阻止 <a> 默认跳转
a.addEventListener('click', (e) => {
  e.preventDefault();
})
.简单粗暴型的，直接在结构上拍死
<a href="javascript:;">link</a>
[] 阻止 <form> 默认提交跳转
form.addEventListener('submit', (e) => {
  e.preventDefault();
})
.或
form.addEventListener('submit', (e) => {
  return false;
})
[] 阻止右键菜单
document.addEventListener('contextmenu', (e) => {
  e.preventDefault();
})
.或
document.oncontextmenu = function () {
  return false;
}
