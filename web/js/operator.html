逻辑运算符 Logical

  • 结果是 Boolean
  • 用于求值
逻辑运算符 Logical
item desc
&&
逻辑与
也可以写作 AND
求值；用做判断时，根据值决定是真还是假
两个值的情况：先将运算符左边的表达式转化为 Boolean，如果为真，就返回右边的值；如果为假就返回左边的值，又称短路，碰到假就停止
多个值的情况：如果左边为真，就继续往后看；如果还为真，就再继续；如果为假，就停止
undefined、null、NaN、""、''、0、false 转化为 Boolean 后，都为 false
运算符两侧的表达式可以是任何类型，如：根据某个条件，指向相应的动作

! 逻辑非

instanceof

  • 检查某个变量是否是 对象 的一个实例
  • 所有引用类型的值都是 Object 的实例
  • 不仅可以知道某个变量是不是对象，还可以知道它是什么类型的对象：Object、Array、RegExp
  • 如果检测基本类型的变量，返回 false；因为基本类型不是对象
console.log(12 instanceof Number);//false
console.log('hi' instanceof String);//false
console.log(true instanceof Boolean);//false
// console.log(null instanceof Null);//warning
// console.log(undefined instanceof Undefined);//warning
console.log(fn instanceof Function);//true
console.log({ name: 'hkc' } instanceof Object);//true
console.log([1, 2] instanceof Array);//true

类型运算符 typeof

  • 检测一个变量是基本数据类型还是引用数据类型
  • 是一个运算符 operator，不是函数
  • 数组 Array、对象 Object 和 null 都是object类型；Object 泛指所有的引用数据类型；如果想知道确切的数据类型，请使用运算符 instanceof
let fn = num => num * 2;
console.log(typeof 12);//number
console.log(typeof 'hi');//string
console.log(typeof true);//boolean
console.log(typeof null);//object；空对象指针
console.log(typeof undefined);//undefined
console.log(typeof fn);//function
console.log(typeof { name: 'hkc' });//object
console.log(typeof [1, 2]);//object

条件运算符 ?:

  • Conditional (ternary) operator
  • 逻辑同if-else；如果真，执行?前面的逻辑；否则执行后面的逻辑
  • 更简洁、高效
12 > 18 ? 'yes' : 'no'//no
12 < 18 ? 'yes' : 'no'//yes

扩展运算符 ...

  • Spread syntax
  • 扩展元素，如 数组、对象赋值
let arr = [1, 2, 3, 4, 5]
let arr2 = [...arr]
