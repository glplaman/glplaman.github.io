颜色

@Color
颜色值/颜色名称
使用颜色名字表示，如 red、green、blue
不建议
RGB 值
rgb(red, green, blue[, alpha])
使用红绿蓝三原色表示颜色
r：red 红色
g：green 绿色
b：blue 蓝色
a：alpha 透明通道；0-1之间某个浮点数；0完全透明；1完全不透明
每个值范围：0-255
常见颜色通常使用 CSS 变量定义
:root {
  --base-color: 255, 68, 0;
  --warn-color: rgba(var(--base-color), 1);
  --warn-color-light: rgba(var(--base-color), 0.05);
}
HEX 值
使用16进制表示；如 #rrggbb
每个颜色的取值范围是 00-ff
同一组颜色，如果相同，可以简写为3位 #rgb，如：#ff4400，可以简写为 #f40
也可以使用第78位/第4位标识一个透明色：#ff440012 最后2位为透明度；#F405 最后1位为透明度
HSL 值
hsl(hue, saturation, lightness[, alpha])
使用色相、饱和度和明度表示颜色
h：hue 色相；是色轮上从 0 到 360 的 度数。0 是红色，120 是绿色，240 是蓝色
s：saturation 饱和度；是一个 百分比值，0％ 表示灰色阴影，而 100％ 是全色
l：lightness 亮度：也是 百分比，0％ 是黑色，50％ 是既不明也不暗，100％是白色
a：alpha 透明通道；0-1之间某个浮点数；0完全透明；1完全不透明
安全色
用 00 33 66 99 CC FF组合的颜色
开始仅仅支持256种颜色
中有40种是保留色，还有216是WEB安全色
用6种红色、6种绿色和6种蓝色任意组合，即：6*6*6=216
