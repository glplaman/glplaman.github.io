  • 元素彼此之间没有顺序；取而代之的是标记 marker；类似于办公文档的项目符号 bullets

初始化 Initialization

Why

  • 列表默认有 margin 和 padding
  • 项目符号或编号会破坏页面布局
  • 保证样式在不同浏览器的一致性

How

  • 使用通配符取消所有元素的 margin 和 padding；并修改盒模型计算方式
  • <ol> 和 <ul>取消默认项目符号和编号；如需要，再根据设计重新指定
  • <dl> 没有更多初始化操作
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

ol,
ul {
  list-style: none;
}

伪元素 ::before

  • 创建一个 inline 元素作为当前元素的第一个 first 子元素；使用 ::after 创建最后一个 last 子元素
  • 必须指定伪元素 content 属性，哪怕为空
  • 可以使用普通字符、CSS实体；也可利用 attr() 获取元素属性
  • content 拼接字符串不能使用 +；直接在后面添加即可
  • content 属性不支持变量使用 var()；可以通过 计数器 迂回实现
  • 应用非常非常非常广泛
  • 更多信息，请访问 伪元素
[ ] 定制无序列表项目符号

. 使用普通字符

li::before {
    content: '>';
}

. 使用 CSS 实体

li::before {
    content: '\21E8';
}

可按照普通元素对其进行定位、布局、旋转等等操作；注意：旋转会撑开/破坏布局，请为父元素设置 overflow 溢出隐藏；同样需要注意的是：他们的 box-sizing 也要初始化为 border-box

伪元素的样式受父级元素的影响；但是伪元素不是元素的直接孩子，无法使用直接后代选择器 > 选择

实体 Entity

  • 实体是一段以连字号 & 开头、以分号 ; 结尾的文本/字符串；是对符号的命名引用
  • 常用于显示保留字符和不可见的字符、或其他难以用标准键盘键入的字符
  • 可以应用在结构或样式中
  • 更多信息，请访问 MDN - Entity、CSS - entities、WHEN TEXT IS NOT ENOUGH

阿里字体图标 Iconfont

特点 Features

  • 矢量图形
  • 减少 HTTP 请求
  • 丰富的图标库
  • 可定制
  • 兼容性好
  • 更多信息，请访问 阿里字体图标 Iconfont

过程 Procedure

  1. 注册|登陆：打开阿里字体图标官网 https://www.iconfont.cn，创建账户
  2. 创建项目；也可以在收集完图标后再创建

    3. . 部分环境，如微信小程序需要 base64 的格式，请在项目设置中勾选

    . 可以修改字体图标前缀和字体家族，默认是 icon- 和 iconfont

  3. 查找图标：搜索并选择字体图标；添加到购物车[免费]；如果需要使用图片，可以直接下载为图片格式
  4. 将购物车中的字体图标添加到项目
  5. 更新并生成链接；可以在线使用，也可以下载到本地使用

    6. . 每次项目中的图标发生变化都要更新链接

  6. 在项目的 CSS 样式文件中引入字体类 Font class - 推荐使用在线链接
    7. @import url(//at.alicdn.com/t/c/font_3859342_u1d9rpqywd8.css);
  7. 在结构中使用 - 指定基类 iconfont 和对应的图标类
    8. <span class="iconfont icon-keyboard"></span>
上传图标至项目：将自己设计的 svg 格式的图标引入；适合设计能力比较强的小伙伴；需要审核

  1. 项目创建 - 略

计数器 Counter

  1. 初始化/设置计数器：全文编号时，通常可以省略；分段编号需要显示指定
  2. 计数增加：元素中指定；哪个元素计数，哪个元素指定；.el 指的是计数元素
    3. .el {
  counter-increment: sn;
}
  3. 使用计数：在计数元素的伪元素 ::before 中指定
    4. .el::before {
  content: counter(sn);
}
[ ] 表编号 - 全文编号
body {
  counter-reset: tbl-sn;// 可略
}

caption {
  counter-increment: tbl-sn;
}

caption::before {
  content: '表' counter(tbl-sn)' ';
}
[ ] 图编号 -全文编号
body {
  counter-reset: fig-sn;// 可略
}

figcaption {
  counter-increment: fig-sn;
}

figcaption::before {
  content: '图' counter(fig-sn)' ';
}
[ ] 参考文献编号 - 分段编号；使用类 .reference 限制编号类型
ol.reference {
  list-style: none;
  counter-reset: list-sn;
}

ol.reference>li {
  counter-increment: sn;
}

ol.reference>li::before {
  content: '[' counter(sn)'] ';
}
