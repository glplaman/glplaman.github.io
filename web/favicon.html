网站图标

@Favicon.ico
Overview
网站图标通常放在网站根目录下
在 <head> 区使用 <link> 引入
网站图标支持多种格式，使用 type 区分
为了确保兼容性，建议提供多个格式的 favicon
浏览器会根据需要选择合适的图标尺寸，也可以显式指定大小，如 size = '32x32'
.ico
默认类型；使用最为广泛
默认名字是 favicon.ico
制作icon的工具很多，如：比特虫、favicon.io
<link rel="icon" type="image/x-icon" href="favicon.ico" >
.svg
可缩放矢量图形，适合高分辨率屏幕，提供更好的清晰度和灵活性
各大平台都有；如阿里字体图标，直接下载并使用
<link rel="icon" type="image/svg+xml" href="xxx.svg">
.png
文件大小较小，加载速度快。
<link rel="icon" type="image/png" href="xxx.png">
.gif
支持动画效果；受大小限制，动画效果体现不是很直接
<link rel="icon" type="image/gif" href="xxx.gif">
.jpeg
<link rel="icon" type="image/jpeg" href="xxx.gif">
.webp
<link rel="icon" type="image/webp" href="xxx.gif">
