概述 Overview
.变形 transform 允许您移动 translate、缩放 scale、旋转 rotate 和倾斜 Skew；综合使用，可以呈现3D效果
.分为2D变形和3D变形，分别有相应的CSS属性
.适用于 block 元素
.可以在不使用定位的情况下，移动元素，且提高层级
.每个变换都可以指定起点 transform-origin ；默认是对象几何中心 - 50% 50% 0
.同时应用多个变换属性时，属性顺序不同，最后结果也不相同；建议先旋转 rotate 再平移 translate
.变换不会引用 reflow 重新布局
.更多变换，请访问 ztext.js
When possible, use transforms and opacity for animation rather than layout properties like "top", "left" or "margin". You'll get smoother animations!

位移 Translate

类别 说明
translateX(tx) X轴位移 - horizontally on the 2D plane
translateY(ty) Y轴位移 - vertically on the 2D plane
translateZ(tz) Z轴位移 - along the z-axis in 3D space
translate(tx [, ty]) 1个参数，沿x轴；2个参数，分别沿着x轴、y轴 - in the horizontal and/or vertical directions
translate3d(tx, ty, tz) 3D位移 - in 3D space
translateX(tx) = translate(tx, 0) = translate3d(tx, 0, 0)
translateY(ty) is equivalent to translate(0, ty) or translate3d(0, ty, 0).
translateZ(tz) is equivalent to translate3d(0, 0, tz)
translate(2px) = translate(2px, 0)

旋转 Rotate
旋转 Rotate
.单位：度 deg、弧度 rad、圈 turn
.方向：正数为顺时针；负数为逆时针 - If positive, the movement will be clockwise; if negative, it will be counter-clockwise.

<section2> 核心属性 Core Property
.transform 默认是2D变形；要呈现3D效果，必须设置透视/景深及指定为3D变形模式；具体步骤为：
1. 为祖先元素指定景深 perspective (创建一个天空盒 sky box)，单位是px，通常取800px-1200px；景深和视口宽度相差不要太大 - start by configuring the 3D space by giving it a perspective 
.super {
    perspective: 1000px;
}
2. 为自己显式的设置3D变形 transform-style: preserve-3d( flat 表示2D) - then you have to configure how your 2D elements will behave in that space
.self {
    transform-style: preserve-3d;
}
这两个属性会影响元素的固定定位

<section4> 注意事项 tips
.父元素不能指定 overflow:hidden；否则就还原为2D模式
.如果为更高级的父元素指定景深 perspective，则其下面子元素都应指定3D变换模式，即：这些元素可以共享一个景深/天空盒(sky box)
.backface-visibility可以隐藏元素背面，和3D并没有直接关系
.更多信息，请访问 MDN - 3D specific CSS properties、发牌、科技环 - CodePen
