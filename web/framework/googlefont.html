基本步骤
点击打开 GoogleFonts
右侧搜索字体，并"选择这个字体"
点击底部的对话框，进一步定制，如 light、regular、medium、bold；可以下载、嵌入
复制 link 到自己的项目样式文件中去
@import url('https://fonts.font.im/css?family=Roboto:300,400,500,700');
使用 link 中提供定制的字体名字，如 Roboto；稳健考虑，每个字体都指定了一个备用的基础字体
font-family: 'Roboto', sans-serif;
根据需要指定权重；默认是 regular 400
h2 {
    font-weight: 700;
}
如果仅仅指定一种权重如300，你仍然可以通过 font-weigh 来在加粗如600；浏览器会尝试模拟加粗效果；所以如果需要粗体，请定制字体时明确指定需要的权重
推荐在线定制后导入 import 字体.CSS文件，方便样式集中管理
