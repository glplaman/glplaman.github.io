步骤
创建 H5 页面
引入库到项目中
<script src="https://unpkg.com/typed.js@2.0.132/dist/typed.umd.js"></script>
准备目标元素
<h2 id="greeting"></h1>
在 JavaScript 中，创建 typed 的构造函数，指定打字的目标元素和打字的内容，以及其它打字的选项，如打字速度、删除速度等等
let typed = new Typed('#greeting', {
  strings: ['hello', 'world'],
  typeSpeed: 100,
  backSpeed: 100,
  backDelay: 1000,
  loop: true,
  loopCount: 3
});
