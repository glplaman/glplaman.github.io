基本使用 Usage

简介 Overview
Java大小写敏感
Java语句以分号;结束
类 Class
每个Java文件都是一个Java类
Java文件的主类名应同文件名保持一致
一个Java文件可以有多个类，但是只能有一个public类，即：主类
主类的入口函数是main()
[]HelloWorld.java
package base.class1;

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello world");
    }
}
关键字 keyword
系统使用的单词，以特殊颜色区分
public、static、final
标识符 Identifier
以字母和_开头，可以包含关键字，但是不能单独使用关键字；语义化[见名知意]
包名：全部小写
类名：大驼峰UpperCamelCase，所有单词首字母大写，如HelloWorld
方法名：小驼峰LowerCamelCase，首单词的首字母小写，后面其他单词首字母大写，如getLength()
变量名：小驼峰LowerCamelCase，如unitId
常量名：全部大写，INCOME_SUM
注释 Comment
单行注释：//；快捷键：CTRL+/
多行注释，也叫块注释：/* */；输入/*并回车，自动生成
快捷键 Shortcut
快速搜索：双击Shift
JavaDoc
以html页面的形式生成Java doc
工具Tools -> Create JavaDoc：指定输出目录；如果有中文，必须指定编码集；这些参数都会在命令中体现出来
常用插件 Plugins
设置Setting -> Plugins里可以添加需要的插件
