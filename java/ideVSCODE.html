[]原生创建 HelloWorld
1. 创建文件夹，用来存放java程序，如JavaPro
2. 创建HelloWorld.java，编辑代码
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("hello,world");
    }
}
3. 点击主函数main上方快捷工具栏 Run|Debug 进行执行或调试；在底部的终端窗口Terminal中可以看到结果
4. 也可以在终端中使用传统方式编译并执行
PS E:\JavaPro> javac .\HelloWorld.java
PS E:\JavaPro> java.exe .\HelloWorld.class
错误: 找不到或无法加载主类 .\HelloWorld.class
原因: java.lang.ClassNotFoundException: /\HelloWorld/class
PS E:\JavaPro> java.exe HelloWorld        
hello,world
[]无框架创建 HelloWorld
1. 打开 vs code，Ctrl+Shift+P，输入或选择 Java:Create Java Project，再选择 No Build Tool
2. 选择一个空文件夹，如Demo，确定后输入项目名字如Demo并回车，重新打开一个vs code编辑器窗口，项目结构如下
Demo
    .vscode
    lib
    src
    App.java
    README.md
3. 暂时不需要App.java，删除
4. 在src下创建多级目录，如base\class1；src就是存放源代码的地方；通常以目录的形式存放不同的java文件
5. 选择class1，创建HelloWorld.java，系统自动创建对应的包package，本质上仍然是以目录的形式组织代码
package base.class1;

public class HelloWorld {

}
6. 完善代码并执行，在窗口底部查看输出结果；输入main并回车，vs code会自动创建对应的代码片段Snippets
package base.class1;

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello world");
    }
}
