基本概念
默认情况下，程序按照代码从上到下的顺序执行
使用流程控制语句可以改变程序的执行方向
分为：
. 顺序结构
. 分支结构：if-else、switch-case
. 循环结构：for、do-wihle、while-do
. 嵌套结构：多种结构的组合使用
if-else
单分支结构
. 单个条件判断
//伪代码
if (关系表达式)
    代码块
双分支结构
. 两个条件判断
//伪代码
if (关系表达式)
    代码块1
else
    代码块2
多分支结构
. 多个条件判断
//伪代码
if (关系表达式1)
    代码块1
else if(关系表达式2)
    代码块2
else
    代码块3
[] 成绩输出
if (score >= 90)
    System.out.println("A");
else if (score >= 80)
    System.out.println("B");
else if (score >= 60)
    System.out.println("C");
else
    System.out.println("Fail");
嵌套if-else结构
. 每个代码块都可以是一个if-else结构
. 条件的多重判断，请小心使用
. 嵌套结构中，else和最近的if匹配，不要迷信格式的结构
说明
. 代码块只有1行代码时，可以略去{}；但是为了增加代码可读性，建议使用{}
. 布尔型数据可以直接使用，无需使用==或!=判断
. 多分支结构因为可读性不好，尽量避免使用，而是使用switch-case结构
. 应合理安排条件，以提高执行效率
switch-case
. 根据 常量表达式 来匹配不同的分支；不可以使用关系表达式
. 使用break结束逻辑，否则会出现case穿透，即判断多个逻辑
. 必须指定default分支；default分支可以不用break，但是不建议
. 应合理安排条件，以提高执行效率
switch (表达式) {
    case val0:
        break;
    case val1:
        break;
    ...
    case valN:
        break;
    default:
        break;
}
[] 成绩输出
switch (score/10) {
    case 7:
    case 8:
    System.out.println("B");
    break;
    case 9:
    System.out.println("A");
    break;
    case 6:
    System.out.println("C");
    break;
    default:
    System.out.println("D");
        break;
}

for
所有的循环结构都包括：
. 初始化循环语句
. 循环判断语句
. 循环迭代语句
. 循环体语句
for( 初始化循环语句; 循环判断语句; 循环迭代语句){
    循环体语句;
}
快捷操作：输入fori按tab，可以快速生成以i为迭代的代码片段
[] 0-100的偶数
for (int i = 0; i <= 100; i++) {
    if (i % 2 == 0) {
        System.out.println(i);
    }
}
[] 求0-100的和
while
while
while( 循环条件语句){
    循环体语句;
}
1. 执行循环条件语句
2. 如果假，结束while循环
3. 如果真，执行循环体语句，并重新判断循环条件语句，直到结束循环
do-while
. 先执行再判断，即：至少执行一次逻辑
. 实际开发中，使用并不多
do{
    循环体语句;
}while( 循环条件语句);
