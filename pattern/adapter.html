适配器模式

Adapter

概述
将一个类的接口转换成客户希望的另一个接口：让不兼容的类一起协调工作
通过定义同名方法 display实现
地图调用
多个地图可用，执行同一个方法
参考代码
class BaiduMap {
  show() {
    console.log('baidu show');
  }
}

class GaodeMap {
  display() {
    console.log('gaode display');
  }
}

class adapterBaidu extends BaiduMap {
  constructor() {
    super()
  }
  display() {
    this.show()
  }
}

function renderMap(map) {
  map.display()
}
renderMap(new adapterBaidu())
renderMap(new GaodeMap())
