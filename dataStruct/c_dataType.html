字符 char
1个字节Byte
使用单引号''将字符扩起来
输出时，使用格式控制符%c
可以存储数字的范围[-128, 127]。如果使用的数字小于128，为了节省空间，可以声明为char
部分字符和数字一一对应，如下表
0 1 ... 8 9 a A B ... X Z b ... y z
48 49 ... 56 57 65 66 ... 89 90 97 98 ... 121 122
执行以下代码，查看结果
#include <stdio.h>
int main()
{
    int i;
    for (i = 48; i <= 127; i++)
    {
        printf("%c\t%d\n", i, i);
    }
    return 0;
}
