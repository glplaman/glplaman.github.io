[ 理论讲解 Lecture on Theory ]

1. 概述 Overview
  • 创建型 Creational 设计模式
  • 一个类仅有一个实例 - single instance
  • 提供一个全局访问点
  • 避免全局使用的类频繁的创建和销毁，占用内存
2. 特性 Feature
  • 唯一实例
  • 全局访问
  • 延迟初始化 - 首次使用时候才实例化
3. 应用 Application
  • 弹出对话框 Modal
  • 数据库连接 Database Connection
  • 状态管理 - Vuex、Pinia
  • 路由管理 - Vue Router
  • 配置管理 Config
  • 日志管理
  • 缓存管理 Logger
  • 国际化 i18n
  • 音频/视频播放器控制器 Player
  • 其它...
4. 实现 Implement
  • 闭包 Closure + 立即执行函数 IIFE

    利用闭包保存 instance 变量，外部无法直接访问

    使用 IIFE 创建私有作用域，instance 成为真正的私有静态变量

    返回一个函数（构造函数），每次调用都检查是否已创建实例

    JavaScript支持较好；低版本也可以正常使用

    语法略显复杂，可读性不如类

    返回的是函数，调用时是 Singleton()，不像构造函数 new Singleton() 直观

    如果初始化逻辑复杂，代码组织不如类清晰

    • const Singleton = (() => {
  let instance;
  
  return () => {
    if (!instance) {
      // ...
    }
    return instance;
  };
})();
  • 类 Class

    在 constructor 中检查 Singleton.instance 是否存在

    若存在，直接返回已有实例，跳过初始化

    否则创建新实例并赋值给 Singleton.instance

    语法清晰，符合现代 JS 风格（使用 class）

    调用方式自然：new Singleton()

    易于扩展，支持继承（但需小心处理单例逻辑）

    Singleton.instance 是暴露的，可能被外部修改或重置。如何避免？

    • class Singleton {
  constructor() {
    if (Singleton.instance) {
      return Singleton.instance;
    }
    
    // 初始化逻辑
    // ...
    Singleton.instance = this;
  }
  
  // methods
  // ...
}
  • 模块 Module - singleton.js

    天然单例：JS 模块系统保证了这一点

    利用 ES Module 的特性：模块是单例的，import 多次也只加载一次

    直接导出一个实例，而不是类

    最简单、最安全：没有手动检查实例的逻辑

    易于测试（可通过 mock 模块替换）

    无法延迟初始化（lazy initialization）：模块加载时就创建实例，即使还没用到

    如果初始化依赖运行时参数（如配置），会受限

    不适合需要“按需创建”的场景

    • class Singleton {
  constructor() {
    this.data = [];
  }
  
  getData() {
    return this.data;
  }
  
  addData(item) {
    this.data.push(item);
  }
}

// 导出单例实例
export default new Singleton();
5. 优势 Merit
  • 节省内存：避免创建多个实例，减少内存占用
  • 全局访问：提供统一的访问点，便于管理共享资源
  • 控制实例化：确保某些类只存在一个实例

  • 首次调用：创建DOM元素并添加到页面；后续调用：直接返回已创建的实例
  • 关闭操作：隐藏模态框而不是销毁
  • 再次打开：直接显示已存在的模态框
  • 性能提升：避免重复的DOM操作和事件绑定

    }
    return instance
  }
})()

[] 模态框 - 进阶版
  • 业务聚合
    • class Modal {
    constructor() {
        if (!Modal.instance) {
            this.element = document.createElement('div');
            this.element.innerHTML = 'Modal';
            this.element.style.display = 'none';
            document.body.appendChild(this.element);
            Modal.instance = this;
        }
        return Modal.instance;
    }

    show() {
        this.element.style.display = 'block';
    }

    hide() {
        this.element.style.display = 'none';
    }
}
