HTML
网页有种类繁多的标签tag组成；标签页叫标记；不同的标记有不同的属性
使用开始标记符号<和结束标记符号>包裹标签名，如<p>
标签和标记符号都是英文字符
大部分标签都是双标记：有开始标记和结束标记
<标记 属性名1="属性值1" 属性名2="属性值2">内容<标记>
先输入一个<，再根据代码提示选择需要的标签，不要一个字母一个字母输入
HTML组成
由<head>和<body>组成
<head>
网页头部信息
由<meta>、<title>、<link>、<script>和<style>组成
内容多不可见，大部分供搜索引擎使用
<body>
网页主题部分|正文部分
由文本标签、媒体标签、链接标签和框架标签等等组成
内容可见：用户看到的、浏览器中显式的就是这部分内容
主体属性
属性 说明
text 正文文字颜色
bgcolor 背景颜色
background 背景图像
topmargin 网页上边距
leftmargin 网页左边距
Tags
标题 <h1> - <h6>
heading
共6级标题，字体大小一次减少
<h1>：一级标题；一个页面只应出现一个一级标题
<h2>：二级标题
<h3>：三级标题
<h4>：四级标题
<h5>：五级标题
<h6>：六级标题
段落 <p>
paragraph
一段文本
属性 align：文本对齐方式
段落对齐属性align取值
属性 说明
left 左对齐，默认
center 居中对齐
right 右对齐
换行 <br>
break
如果不想另外新起一个段落，可以使用换行标记
文本 <font>
设置文本的字体类型、大小和颜色
字体属性
属性 说明

可以指定多个字体，字体之间使用,分割
尽量使用宋体和黑体等常见字体
size 字体大小
像素px，如16px
也可使用字号1-7，1最小字号；7最大字号
color 字体颜色
通常使用16进制表示，如#ff4400
黑体 <b>
bold

斜体 <i>
italic

下划线 <u>
underline
字体带下划线

水平分割线属性
属性 说明

align 线条对齐方式，同段落<p>
color 线条颜色

image
定义图片
图片属性
属性 说明
alt 图像的提示信息；alternative
src 图像的URL，可以是相对路径；也可以是绝对路径；source
align 图像的对齐方式

超链接 <a>
anchor

可以使用普通文字；也可以使用其他元素，如图片

属性 说明
href 链接的地址
url 资源地址；可以是网址或某个PDF文件

表单 <form>
定义表单，进行数据提交
input分类
<input> 类型 说明
text 输入单行文本
password 输入密码
radio 单选按钮；多个选一
checkbox 多选按钮；多个选一

value 值

属性 说明

cols 文本区的宽度
wrap 是否换行；默认换行
off：不允许换行

Script
说明
脚本的参与使得网页从静态变为可以实现人机交互的动态页面
赋予HTML元素一定的行为能力
脚本需要借助标签<script>嵌入到HTML中
支持的脚本有VBScript、JavaScript。默认是JavaScript，可以不写
需要指明脚本语言类型
早期HTML开发中，脚本可以写在HTML任何位置
现在的HTML5开发规范是三分离：结构HTML、样式CSS和逻辑JavaScript互相分离
在HTML5中以外部文件的形式引入样式文件.css和逻辑文件.js
