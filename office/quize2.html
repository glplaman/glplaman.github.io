目的
. 分类汇总的使用

分类汇总
. 统计各职称教师的平均工资
打开教师资料
按照"职称编码"或"职称"排序
点击"分类汇总"，分类字段选择"职称"；汇总方式选择"平均值"，汇总项选择"工资"，其它保持默认并确定
图表
. 先根据"学历"分类汇总，计数，再提取汇总数据生成图表
学历计数的分类汇总操作同平均工资
点击汇总级别"2"，仅显示汇总数据
隐藏不需要的数据列[可选，反正后面要调整]
框选汇总数据和学历数据；不需要标题，复制可见单元格

方法1："开始" → "查找和选择" → "定位条件" → "可见单元格" → CTRL + C复制刚才选中的数据 → CTRL + V粘贴到新的单元格或工作表

方法2：Ctrl + G → "定位条件" → "可见单元格" → 复制并粘贴

方法3：F5 → "定位条件" → "可见单元格" → 复制并粘贴

方法4：按ALT + ; → 复制再粘贴

调整列顺序：选择人数所在列，按住Shift，鼠标置于列交界处，拖动到学历列的另外一侧完成交换；或直接复制调整数据位置
增加标题：学历和人数
定位鼠标在当前数据区域，"插入" → "饼图" → "三维饼图"
"+" → 增加"数据标签"，并调整数据标签在外
调整"图例"到"顶部"
右键单击绘图区域的数据系列 → "设置数据标签格式" → "百分比"
右键单击绘图区域的数据系列 → "设置数据系列格式" → "饼图分离" → 10%
为饼图应用"图表样式"中的"样式7"
使用"移动图表"将饼图移动到新工作表"学历结构饼图"
Tips
充分利用图表的动态选项卡设置图表
复制可见单元格的四种方法
