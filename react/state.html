Introduction

  • 响应式数据

useState()

  • 状态：state，即数据
  • 函数：function，修改数据
  • 命名：原则上任意，建议 something, setSomething
  1. 状态是基本类型时，应该直接传递新值
    2. const [show,setShow] = useState(false)
function hansectioneClick(){
  setShow( true )
}
  2. 状态是对象
    3. const [showState,setShowState] = useState({show: false})
function hansectioneClick(){
  setShowState( {show: true} )
}
