响应状态码

@Code
服务器会根据响应的处理结果，自动返回状态码
也可以显式的指定状态码，而不管和处理结果是否匹配；如果状态码不符合一般规定，使用开发者工具检查时，会提示未知 unknown
res.statusCode = 200
100 - 199
信息响应
200 - 299
成功响应：请求处理正常
300 - 399
重定向消息
301 - 资源转移
304 - 数据缓存，条件没有发送变化时，拉取本地缓存，无须再次远端拉取
CTRL +F5强制刷新后，成功后，状态码为200；再次刷新，变为304
400 - 499
客户端 / C端响应错误；常见的有 404
500 - 599
服务器端 / S端响应错误
