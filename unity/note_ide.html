你好世界 hello world
1.创建项目-选择Projects→New Project
创建项目
2.创建3D项目-选择项目类型、名字和位置，不选择版本控制[需要git支持，方便代码管理，感兴趣的同学，可以查看本网其它页面或自行百度]
创建项目
3.项目创建中[视电脑配置情况，时间可能比较久。。。每次上课前，请提前准备好环境]
创建项目中
4.项目创建完毕，进入主界面
项目主界面
5.打开菜单编辑Edit，选择界面Preferences，再选择外部工具 External Tools，指定为Vs Studio；如果安装了Vs Code也可以选择使用，下载相应的插件即可
关联Vs Studio
6.在层级Hierarchy空白处，右键单击，创建一个空对象Create Empty
7.在项目Project空白处，右键单击创建一个脚本，命名为Demo，后缀名默认是.cs，可以不输入
8.将脚本拖动到空对象上，使两者关联在一起
9.双击脚本文件，系统会根据之前的关联，打开Vs Studio；可以看到系统默认添加了很多基础代码；在Start中输入print("hi,there");保存并返回Unity
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;
using UnityEngine.SceneManagement;
public class demo : MonoBehaviour
{
  // Start is called before the first frame update
  void Start()
  {
    print("hi,there");//用户输入
  }

  // Update is called once per frame
  void Update()
  {
    transform.Rotate(0, 1, 0);
  }

}
10.点击播放按钮，项目开始编译调试；完毕后，在底部控制台可以看到输出的测试信息
