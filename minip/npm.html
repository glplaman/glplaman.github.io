@NPM
. 需要 Node.js 环境支持

Steps

  1. 安装依赖
  2. 构建 NPM；构建完毕后，目录 miniprogram 变为 miniprogram_npm
  3. 引入包
  4. 使用包

手动构建 NPM

  • 打开项目配置文件 project.config.json，修改相关字段如下
  • 重新启动开发者工具
  • 执行构建
    • "packNpmManually": true,
"packNpmRelationList": [
  {
    "packageJsonPath": "./package.json",
    "miniprogramNpmDistDir": "./miniprogram/"
  }
],

  1. 安装依赖

    2. . 在小程序根目录，右键 → 打开内建终端|小黑屋
