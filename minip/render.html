从WebView迁移到Skyline
1. json设置

app.json增加配置项

"renderer": "skyline",
"rendererOptions": {
    "skyline": {
        "defaultDisplayBlock": false
    }
}

页面.json增加配置项 - 不需设置导航标题，也无法设置

"navigationStyle": "custom",
"disableScroll": true
2. 项目设置
勾选"详情" → "本地设置" → "启用Skyline渲染调试"
3. 设置完毕后，模拟器顶部出现"当前渲染模式 Skyline"，表示迁移成功
