声明式导导航

<navigator>

简介 Introduction

  • 页面之间的导航/链接，类似HTML的 <a>
  • 使用 <navigator> 实现的导航，称为声明式导航
  • 跳转到普通页面
  • 跳转到标签页
  • 跳转到外部小程序
  • 还可以指定返回的层级
  • 可以使用文字；也可以使用图片等其他元素
  • 除了声明式导航，还可以利用导航组件对应的 API，实现 编程式导航
  • 更多信息，请访问 <navigator>
导航组件属性
item desc
target 打开位置
self：当前小程序；默认
miniProgram：外部小程序
url 跳转链接|地址；相对地址或绝对地址
open-type 跳转类型
navigate：默认，也可以是外部小程序
switchTab：到标签页
delta 数字，后退的层级；默认是1，可以省略
app-id 外部小程序的 AppID；在"更多"或"设置"或公众号关联的小程序中获取
path 外部小程序的页面；默认是主页
extra-data 传递给外部小程序的数据
hover-class 指定点击时的样式类；none 时，没有点击态效果；或修改默认的导航类样式 navigator-hover

无参导航

  • 跳转到小程序内部普通页面 - 仅指定 url
    • <navigator url="../test/test">To Page Test</navigator>
  • 页面层级比较深，可以使用根路径，从 pages 开始
    • <navigator url="/pages/test/test">To Page Test</navigator>
  • 跳转到外部小程序 - 指定 app-id 和 target
    • <navigator app-id="wx91d27dbf599dff74" target="miniProgram">To JD</navigator>
  • 跳转到标签页 - 指定 open-type
    • <navigator url="../photo/photo" open-type="switchTab">To Tab Photo </navigator>
  • 后退1页 - 省略 delta
    • <navigator open-type="navigate">Back</navigator>

带参导航

  1. 拼接参数 - 在路径 url 后面以值对 key-value 的形式拼接参数

    2. 参数和 url 之间使用 ? 连接

    多个参数以 & 分隔

    静态数据 - url?name=value&...

    <navigator url="../test/test?name=gl">To Page Test</navigator>

    动态数据 - 使用数据绑定

    <navigator url="../test/test?name={{uname}}">To Page Test</navigator>
  2. 目标页获取参数 - 传递的参数在目标页的生命周期函数 onLoad 中通过形参 options 获取
    3. onLoad(options) {
    console.log(options);
}

  • 使用 <navigator>，修改个人中心静态页面，采用列表渲染定制导航；根据不同的导航项，动态设置目标页的导航标题
  • 该案例也适合客户、产品详情页，如：使用一个页面展示不同的商品内容

    <text>团队</text>
</navigator>
  • 移除点击状态：为 <navigator> 指定 hover-class 为 none，无效，修改 默认的导航类
  • 默认样式：带背景色，降低透明度
    • .navigator-hover {
  background-color: rgba(0, 0, 0, 0.1);
  opacity: 0.7;
}
  • 修改为：移除背景色；不改变透明度
    • .navigator-hover {
  background-color: rgba(0, 0, 0, 0.0);
  opacity: 1;
}
