收货地址

@chooseAddress
  • 获取用户收货地址。调起用户编辑收货地址原生界面，并在编辑完成后返回用户选择的地址
  • 需要在 app.json 中进行声明，否则将无法正常使用该接口
  • 更多信息，请访问 wx.chooseAddress(Object object) 
    • "requiredPrivateInfos":[
    "chooseAddress"
]
常用属性
item desc
userName 用户姓名
telNumber 用户电话
provinceName 省
cityName 市
countyName 县
detailInfo 详细地址
postalCode 邮编
[] 获取用户地址
